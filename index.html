CSS3

Apa itu CSS? CSS merupakan singkatan dari Cascading Style Sheets, yaitu bahasa standar yang digunakan untuk menggambarkan presentasi (tata letak dan pemformatan) dari halaman web. Sebelum CSS, hampir semua atribut penyajian dokumen HTML terkandung dalam markup HTML.
